# AND101 Project 3 - AnimalApps

Submitted by: **Evelise Guenda**  
Time spent: **12 hours**

---

## 📱 Summary

**AnimalApps** is an Android app that recreates a screen from the Duolingo app — reimagined with an animal theme. The project focuses on building polished layouts using `ConstraintLayout`, `ImageView`, and `TextView` components, along with a bonus original app layout concept.

If I had to describe this project in three emojis: ☺️❤️😍

---

## ✅ Features

- Duplicated a Duolingo app screen with an Animal theme
- Built a fully runnable Android app
- Layout built with `ConstraintLayout`, `ImageView`, and multiple `TextView` components
- Created a second original layout — a home/onboarding screen for a personal app idea
